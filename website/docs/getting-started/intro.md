# Getting Started with Actuate-AI

This guide will help you get started with the Actuate-AI framework for physical AI and humanoid robotics applications.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js v18 or higher
- npm package manager
- Git for version control

## Installation

To install Actuate-AI, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/abdullah/Actuate-AI.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Actuate-AI
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Documentation Site

To run the documentation site locally:

```bash
cd website
npm run start
```

The site will be available at `http://localhost:3000`.
