# Tutorials

This section contains hands-on tutorials that will help you apply the concepts of physical AI and humanoid robotics.

- Setting up your first humanoid robot simulation
- Implementing basic locomotion patterns
- Sensor integration and processing

Each tutorial builds on the concepts introduced in the [Core Concepts](../concepts/intro.md) section, so make sure you're familiar with those fundamentals before diving into the practical exercises.
