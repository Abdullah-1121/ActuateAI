# Data Model: Docusaurus Setup

## Docusaurus Project
- **name**: Project name (string)
- **version**: Docusaurus version (string)
- **configuration**: Site configuration object
- **dependencies**: List of npm packages (array of objects)
- **content**: Documentation files and assets (object)
- **metadata**: Project metadata (object)

## Configuration File (docusaurus.config.js)
- **title**: Site title (string)
- **tagline**: Site tagline (string)
- **url**: Production URL (string)
- **baseUrl**: Base URL for the site (string)
- **favicon**: Path to favicon (string)
- **organizationName**: GitHub organization/user name (string)
- **projectName**: GitHub repository name (string)
- **deploymentBranch**: Branch for GitHub Pages deployment (string)
- **trailingSlash**: Whether to append trailing slashes to URLs (boolean)
- **i18n**: Internationalization configuration (object)
- **presets**: Docusaurus presets configuration (array)
- **themes**: Additional themes configuration (array)
- **plugins**: Additional plugins configuration (array)
- **themeConfig**: Theme-specific configuration (object)
  - **navbar**: Navigation bar configuration
    - **title**: Navbar title (string)
    - **logo**: Logo configuration
      - **alt**: Logo alt text (string)
      - **src**: Logo source path (string)
    - **items**: Navigation items (array of objects)
  - **footer**: Footer configuration
    - **style**: Footer style (string)
    - **links**: Footer links (array of objects)
    - **copyright**: Copyright text (string)

## Documentation Content (docs/)
- **id**: Unique document identifier (string)
- **title**: Document title (string)
- **sidebar_label**: Label for sidebar navigation (string)
- **sidebar_position**: Position in sidebar (number)
- **description**: Document description (string)
- **tags**: Document tags (array of strings)
- **draft**: Whether document is a draft (boolean)
- **authors**: Document authors (array of objects)
- **slug**: Custom URL slug (string)
- **custom_edit_url**: Custom edit URL (string)
- **parse_number_prefixes**: Whether to parse number prefixes (boolean)
- **content**: Markdown content (string)
- **metadata**: Additional document properties (object)

## Sidebar Configuration (sidebars.js)
- **type**: Sidebar item type (string - 'category', 'doc', 'link', etc.)
- **label**: Sidebar item label (string)
- **items**: Child items (array of sidebar items)
- **link**: Link configuration (object)
  - **type**: Link type (string - 'doc', 'generated-index', etc.)
  - **id**: Document ID for doc links (string)
  - **title**: Page title for generated index (string)
  - **description**: Page description for generated index (string)
  - **slug**: Custom slug for generated index (string)

## Static Assets (static/)
- **files**: Static files (object)
  - **path**: File path (string)
  - **type**: File type (string)
  - **size**: File size (number)
  - **metadata**: Additional file metadata (object)

## Package Dependencies (package.json)
- **name**: Package name (string)
- **version**: Package version (string)
- **description**: Package description (string)
- **scripts**: Build scripts (object)
- **dependencies**: Runtime dependencies (object)
- **devDependencies**: Development dependencies (object)
- **browserslist**: Browser compatibility targets (array)
- **engines**: Required engine versions (object)

## Navigation Items
- **type**: Item type (string - 'doc', 'docSidebar', 'link', 'dropdown', 'html')
- **position**: Position in navigation (string - 'left', 'right')
- **label**: Navigation label (string)
- **to**: Internal link path (string)
- **href**: External link URL (string)
- **activeBasePath**: Base path for active state (string)
- **docsSideBarId**: Sidebar ID for docSidebar type (string)
- **items**: Dropdown items (array of navigation items)
